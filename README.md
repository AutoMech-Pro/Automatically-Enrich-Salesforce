# Automatically-Enrich-Salesforce-Accounts-with-Web-Crawling-LinkedIn-Data-GPT
Turn raw leads into gold with Web Crawling, LinkedIn Data, and GPT-4 intelligence.


### **Plan Summary**
1.  **Narrative:** The infographic follows the "Data Pipeline" story: *Problem (Manual Work) -> Solution (n8n Architecture) -> Deep Dive (The 4 Phases) -> Result (Efficiency).*
2.  **Visualizations:**
    * **Efficiency Comparison (Bar Chart):** Contrasting "Manual Research Time" vs. "n8n Automation Time" to immediately hook the user with ROI.
    * **Workflow Architecture (HTML/CSS Flowchart):** A custom-built, responsive CSS grid/flex diagram representing the 4-step logic (Trigger -> Discovery -> Intelligence -> Action), strictly replacing Mermaid/SVG with styled `div`s and Unicode arrows.
    * **Enrichment Method Trade-off (Radar Chart):** Comparing "Method A: Search & Infer" vs. "Method B: Deep Scrape" on axes like Cost, Speed, Depth, and Complexity.
    * **AI Data Extraction (Doughnut Chart):** Visualizing the composition of data fields extracted by GPT-4 (Industry, Size, Tech Stack, etc.) to show what the user gains.
3.  **Palette:** "Neon Cyber-Blue". A vibrant, high-contrast palette suited for a tech/automation topic.
    * Primary: `#2563EB` (Bright Blue)
    * Secondary: `#F43F5E` (Vibrant Pink/Red for accents)
    * Background: `#F8FAFC` (Clean Slate) with Dark Cards `#1E293B` for contrast in specific sections.
    * Chart Colors: Derived from this palette.

4.  **Compliance:** No SVG, No Mermaid, Chart.js used with label wrapping and tooltip fixes, Tailwind for styling.
